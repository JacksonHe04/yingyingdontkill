# 结构

- 顶部导航栏
    - 左侧
        - 头像
        - 我当前所在的地点
            - 计算用户到该地点的距离
        - 年龄
            - 类比王者荣耀的游戏等级，进度条为“今年过去的天数/今年的总天数”
    - 中间
        - 灵动岛
            - 状态
            - AI 问答助手小缨缨
                - 点击可输入问题，发送后对话框出现在页面右下角作为悬浮窗口
                - 预置问题：用户输入MBTI和星座测试与作者的匹配度
    - 右侧
        - 导航项
        - 通知
        - 时间
    - Hover 后均会放大
- 最左侧
    - 当前页面的导航栏，点击滚动到对应区域
- 主区域
    - <有大量空位>
- 标签墙
    - 左右水平滚动的多行大量标签小卡片，用户可以点击卡片以肯定
- 留言
    - 卡片在白板中
    - 快捷问候
    - 快速登录
- 底部元信息
    - 访问次数
    - 网站运行时长
    - 网站技术栈
    - 设计理念
    - 更新日期
    - 致谢
- 深水区（彩蛋，达到底部后继续下拉到一定位置进入）
    - <有空位>
