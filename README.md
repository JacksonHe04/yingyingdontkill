# Yingying的数字花园

一个记录创作、技术与生活思考的个人展示网站。

## 项目简介

这是一个基于 Next.js 15 构建的个人展示网站，旨在全面、快速地展示个人信息、经历和能力。网站采用创新的交互设计和视觉效果，结合了毛玻璃设计、3D效果、动画等现代Web技术。

## 技术栈

- **框架**: Next.js 15 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS 4
- **3D渲染**: Three.js + React Three Fiber
- **动画**: Framer Motion
- **数据源**: JSON (readme.json)

## 功能特性

### 核心功能

1. **顶部导航栏**
   - 头像和当前位置
   - 距离计算（基于用户地理位置）
   - 年龄等级系统（类似游戏等级，显示年度进度）
   - 灵动岛AI助手（小缨缨）
   - 平台链接
   - 通知系统
   - 实时时间显示

2. **左侧导航**
   - 页面内锚点导航
   - 滚动时自动高亮当前区域
   - 平滑滚动效果

3. **内容区域**
   - 基础信息
   - 生活信息
   - 经历（照片墙布局）
   - 教育（2D游戏场景占位）
   - 工作（2D游戏场景占位）
   - 开发（不规则卡片布局）
   - 产品（书桌场景占位）
   - 读书（书桌场景占位）
   - 影片（书桌场景占位）
   - 创作（多星球切换占位）
   - 音乐（2行水平排列）
   - 嘻哈（2行水平排列）
   - 活动（不规则卡片布局）
   - 标签墙（可点击点亮）
   - 联系信息
   - 留言板
   - 底部元信息
   - 深水区（思想观点，彩蛋区域）

### 设计特色

- **毛玻璃效果**: 半透明背景 + 背景模糊
- **视差滚动**: 增强视觉层次感
- **流畅动画**: 使用 Framer Motion 实现
- **响应式设计**: 适配各种设备尺寸
- **暗色模式**: 支持系统主题切换

## 开发指南

### 安装依赖

```bash
pnpm install
```

### 开发模式

```bash
pnpm dev
```

访问 [http://localhost:3000](http://localhost:3000)

### 构建生产版本

```bash
pnpm build
```

### 启动生产服务器

```bash
pnpm start
```

## 数据源

所有内容数据存储在 `app/data/readme.json` 文件中，采用结构化的JSON格式。可以根据需要修改该文件来更新网站内容。
